## Install

``git clone https://github.com/codesshaman/pfxer.git`` 
``cd pfxer``

``chmod a+x pfxer``

``sudo cp pfxer /usr/local/bin``

or

``wget https://github.com/codesshaman/pfxer/blob/main/pfxer``

``chmod a+x pfxer``

``sudo cp pfxer /usr/local/bin``

## Usage

``pfxer /path/to/certificate.pfx``
